display: flex,  задается для флекс контейнера
Флекс-элементы всегда располагаются вдоль главной оси, независимо от её направления.
Поперечная ось всегда перпендикулярна главной оси и поворачивается вместе с ней.
Поперечная ось никогда не смотрит вверх или влево. А свойства для поворота поперечной оси нет.
но с flex-wrap:wrap-reverse направление можно поменять наоброт,но отступы другие будут
flex-direction - меняет направление главной оси:  задается для флекс контейнера
    row — значение по умолчанию, главная ось направлена слева направо.
    column — главная ось направлена сверху вниз.
	row-reverse — главная ось направлена справа налево.
    column-reverse — главная ось направлена снизу вверх.

justify-content - распределяет элементы вдоль главной оси:  задается для флекс контейнера
	flex-start — по умолчанию, элементы располагаются у начала главной оси.
	flex-end —  элементы располагаются в конце главной оси.
	center — элементы располагаются по центру главной оси.
	space-between — расстояния между соседними элементами одинаковые, между элементами и краями флекс-контейнера отступов нет.
	space-around — расстояния между соседними элементами одинаковые, между элементами и краями флекс-контейнера есть отступ, равный половине расстояния между соседними элементами.

align-items - выравнивает элементы вдоль поперечной оси:  задается для флекс контейнера
	stretch — по умолчанию, флекс-элементы и растягиваются на всю «высоту» флекс-контейнера. Если флекс-элементам задана высота, то растягиваться они не будут.
	flex-start — элементы располагаются у начала поперечной оси.
	flex-end —  элементы располагаются в конце поперечной оси.
	center — элементы располагаются по центру поперечной оси.
	baseline — флекс-элементы будут выравниваться по базовой линии текста в них, они выстроятся так, чтобы текст в них был как бы на «одной строке».

align-self - выравнивает элемент вдоль поперечной оси:  задается для флекс элемента
	stretch — по умолчанию, флекс-элемент и растягивается на всю «высоту» флекс-контейнера. Если флекс-элементу задана высота, то растягиваться он не будет.
	flex-start — элемент располагается у начала поперечной оси.
	flex-end — элемент располагается в конце поперечной оси.
	center — элемент располагается по центру поперечной оси.
	baseline — флекс-элемент будет выравниваться по базовой линии текста в нем.

flex-wrap - Перенос флекс-элементов:  задается для флекс контейнера
	nowrap — по умолчанию, перенос флекс-элементов на новую строку запрещён.
	wrap — перенос флекс-элементов на новую строку разрешён, если не вмещаются. Ряды элементов располагаются вдоль поперечной оси. Первый ряд располагается в начале поперечной оси, а последний в конце.
	wrap-reverse — элементы будут переноситься, а ряды будут располагаться в обратном порядке: первый в конце поперечной оси, а последний в начале.

align-content - выравниваниет ряды флекс-элементов вдоль поперечной оси, а не на отдельные элементы:  задается для флекс контейнера
	Если есть только один ряд флекс-элементов, то работает align-items.
	Если есть несколько рядов, то работает align-content.
	flex-start — располагает ряды в начале поперечной оси.
	flex-end — располагает ряды в конце поперечной оси.
	center — располагает ряды в середине поперечной оси.
	space-between — равномерно распределяет ряды вдоль поперечной оси, расстояния между соседними рядами одинаковые, отступов у краёв нет.
	space-around — равномерно распределяет ряды вдоль поперечной оси, расстояния между соседними рядами одинаковые, отступы у краёв равны половине расстояния между соседними рядами.
	stretch — по умолчанию, растягивает ряды по высоте.

order - Порядковый номер флекс-элемента, меняет порядок следования флекс-элементов в потоке, не меняя HTML-код:  задается для флекс элемента
	0 — по умолчанию
